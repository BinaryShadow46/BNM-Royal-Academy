# BNM Royal Academy - School Management System

A comprehensive school management system for BNM Royal Academy built with Flask and MySQL.

## Features

### Admin Module
- Dashboard with analytics
- Student management (Add/Edit/Delete)
- Teacher management
- Course management
- Fee management
- Report generation

### Teacher Module
- Student attendance marking
- Grade entry and management
- Timetable view
- Student performance tracking

### Student Module
- Personal dashboard
- Attendance view
- Grade viewing
- Fee status
- Timetable access

### Parent Module (Coming Soon)
- Child's performance tracking
- Fee payment
- Communication with teachers
